mbGDML

Many-Body Gradient Domain Machine Learning

Create, use, and analyze many-body gradient domain machine learning force fields.

Hosted on GitHub

Skills:

Python / Machine Learning

Atomistic insight is fundamental for computational predictive studies of chemical and physical processes. Machine learning force fields provide a route to high-level ab initio calculations at a fraction of the cost. Symmetric gradient domain machine learning (sGDML), a kernel-based method, directly learns the relationship between atomic coordinates and interatomic forces. However, training in the gradient domain sacrifices generalized transferability to other species or number of atoms.

Many-body GDML (mbGDML), is a technique for sGDML transferability to n-sized systems by using many-body machine learning models. Every aspect of the process from preparing ORCA calculations, data set creation, training, and use of mbGDML force fields is taken care of in this user-friendly Python package.

Clusters Data Set

Solvent and Solute-Solvent Clusters Data Set

An open data set containing solvent and solute-solvent cluster structures and energies.

Hosted on GitHub

Created by:

Alex Maldonado

Published:

1/2021

Skills:

Quantum Chemistry / Solvation

This work provides a database of solute, solvent, and solute-solvent clusters and energies. Clusters are curated from literature or from global optimization codes. ABCluster is typically used with the CHARMM force field to generate several low-energy structures. ORCA is then used to optimize structures and calculate electronic energies and free energy corrections. This data set is not exhaustive; data is included as the Keith group's research interests evolve.

Quantum Chemistry

Ab initio modeling provides fundamental atomistic insight into a broad range of chemical, physical, and biological phenomena. I use an assort of density functional theory and wave function theory methods in ORCA and Gaussian.

Solvated Reaction Mechanisms

The vast majority of chemistry occurs in a solvent. I use a variety of solvent models (e.g., implicit and cluster-continuum) along with chain-of-state methods (e.g., growing string method) to elucidate reaction mechanisms.
