Solvent and solute-solvent cluster structures and calculations

Created by:

Alex Maldonado

Duration:

1/2021 - Present

Topics:

Solvation / Data Set / Quantum Chemistry

This work provides a database of solute, solvent, and solute-solvent clusters and energies. Clusters are curated from literature or from global optimization codes. ABCluster is typically used with the CHARMM force field to generate several low-energy structures. ORCA is then used to optimize structures and calculate electronic energies and free energy corrections. This data set is not exhaustive; data is included as the Keith group's research interests evolve.
